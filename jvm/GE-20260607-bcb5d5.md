## Symptom

`NullPointerException` thrown from `Map.copyOf()` in a record compact constructor, with no indication of which entry in the map caused it. A preceding validation call said nothing.

## Context

A Java record validates a `Map<EnumType, String>` field in its compact constructor. A null value is present for one of the map's keys. The code calls `validateOptional()` before `Map.copyOf()`.

## Root Cause

`AgentDescriptorValidator.validateOptional()` (and any similar "optional" validator) returns immediately when the value is `null` ŌĆö by design, null means "not provided" for optional fields. But a null _value_ inside a non-null map is a caller bug, not an absent field. The validator treats it as valid. Then `Map.copyOf()` throws `NullPointerException` because the `Map.copyOf()` contract forbids null keys or values.

The symptom misleads: validation appears to succeed, then an unrelated-looking NPE fires during the copy.

## Fix

Use `validateRequired()` (or an explicit null check with a descriptive exception) for map values. `validateRequired()` delegates to `validateField()` which throws `AgentValidationException("<fieldName>", "must not be null")` _before_ `Map.copyOf()` runs:

```java
if (axisVocabularies != null) {
    axisVocabularies.forEach((axis, uri) ->
        AgentDescriptorValidator.validateRequired(
            "axisVocabularies[" + axis.name() + "]", uri, MAX_VOCABULARY_URI));
    axisVocabularies = Map.copyOf(axisVocabularies);
}
```

**Pattern:** always validate-then-copy; never copy-then-discover. The null check must precede `Map.copyOf()`, and it must use a validator that treats null as an error, not as "absent."

## Why This Is Non-Obvious

`validateOptional` sounds like it validates map values that _may_ be absent. But "optional" means the whole field may be null ŌĆö it says nothing about the contents of a non-null map. A null entry inside a present map is always a bug. The NPE from `Map.copyOf()` points to the copy, not to the null-providing call site, making the chain hard to trace.
