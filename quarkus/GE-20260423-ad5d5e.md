## Symptom

A QA/dev endpoint that seeds game state appears to work (returns 200, no errors) but the seeded units never appear in the UI. Instead, the UI shows state from a completely different source ŌĆö in this case, AI-placed units at unexpected positions.

## Context

A `ShowcaseResource` JAX-RS endpoint injects `SimulatedGame` and calls `simulatedGame.reset()` + `simulatedGame.spawnEnemyUnit(...)`, then `engine.observe()`. In mock mode (`%mock`) this works correctly. In emulated mode (`%emulated`), `SC2Engine` resolves to `EmulatedEngine`, which reads from `EmulatedGame` ŌĆö a completely separate CDI bean. The injected `SimulatedGame` is still available (it's `@ApplicationScoped` with no profile restriction) but the engine never reads from it.

## What was tried

Adjusting unit positions thinking they were in the fog. Recalculating visibility ranges. Restarting the server. None of these revealed the root cause because the engine was broadcasting the wrong game's state all along.

## Root cause

`@Inject SC2Engine engine` resolves to different implementations per profile. The QA endpoint was written assuming mock mode and seeds `SimulatedGame`. Switching to emulated mode swaps the engine implementation without any compile error or runtime warning ŌĆö `SimulatedGame` is still happily seeded but nobody reads it.

## Fix

Document that the endpoint only works in mock mode. Add a comment to the resource class and to CLAUDE.md:

```java
// ShowcaseResource seeds SimulatedGame.
// In emulated mode, EmulatedEngine broadcasts EmulatedGame ŌĆö a separate object.
// This endpoint only works correctly with the default (mock) profile.
```

Run with `mvn quarkus:dev` (no `-Dquarkus.profile=emulated`) before seeding.

## Why this is non-obvious

Both beans exist in every profile. No `@UnlessBuildProfile` annotation excludes `SimulatedGame` in emulated mode. The endpoint returns 200 and the engine's `observe()` call succeeds ŌĆö it just broadcasts the other game's state. There is no log warning, no exception, no indication that the seeded state was ignored.
