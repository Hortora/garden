## What it is

When Quarkus starts the test server for a `@QuarkusTest`, it automatically sets the `test.url` MicroProfile Config property to the full URL of the bound server — including the actual port, even when `quarkus.http.test-port=0` assigns a random one.

```java
// In @BeforeAll (runs after Quarkus server starts):
String baseUrl = ConfigProvider.getConfig().getValue("test.url", String.class);
// → "http://localhost:54321" (actual random port)
```

## Where to use it

Any test infrastructure that needs the full server URL, including:

- **Playwright base classes** — when navigating to pages in `@QuarkusTest` E2E tests
- **Custom REST clients** — when building URLs manually instead of using `@TestHTTPEndpoint`
- **WebSocket test clients** — for constructing `ws://` URLs

## Why it's non-obvious

The official Quarkus testing docs focus on `@TestHTTPEndpoint`, `@TestHTTPResource`, and RestAssured (which auto-configures). `test.url` is not mentioned in the Quarkus testing guide or the configuration reference. It's only discoverable by reading Quarkus test framework internals or by noticing it appears in `ConfigProvider` at test time.

RestAssured works without it because Quarkus auto-configures `RestAssured.port`. Browser-based test infrastructure (Playwright, Selenium) has no such auto-configuration and must resolve the URL manually — making `test.url` essential and undiscoverable from the docs.

## Contrast with alternatives

| Approach | Works? | Notes |
|---|---|---|
| `"http://localhost:8081"` hardcoded | Sometimes | Breaks when random port ≠ 8081 |
| `System.getProperty("quarkus.http.port")` | Sometimes | System property not always set |
| `ConfigProvider.getConfig().getValue("test.url")` | Always | Quarkus guarantees this in @QuarkusTest |

## Important: must be read in @BeforeAll, not at field-init time

```java
// WRONG — reads before server starts, will fail or return null:
protected static final String BASE_URL = 
    ConfigProvider.getConfig().getValue("test.url", String.class);

// CORRECT — @BeforeAll runs after Quarkus server startup:
protected static String BASE_URL;

@BeforeAll
static void setup() {
    BASE_URL = ConfigProvider.getConfig().getValue("test.url", String.class);
}
```
