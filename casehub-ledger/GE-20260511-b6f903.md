### What's undocumented
`LedgerEntryRepository.save(entry)` requires 7 base fields to be manually populated by the caller before calling `save()`. No builder, factory, or documentation states which fields are required or what values to supply. Missing any of these results in a DB constraint violation at runtime.

The complete required field set, discovered by reading `LedgerPrivacyWiringIT.java`:

```java
AdverseEventLedgerEntry entry = new AdverseEventLedgerEntry();
entry.id = UUID.randomUUID();             // required — no @PrePersist auto-assignment
entry.subjectId = domainEntity.id;        // the aggregate being audited; scopes the sequence chain
entry.sequenceNumber = nextSeq;           // 1-based per subject; caller computes this
entry.entryType = LedgerEntryType.EVENT;  // EVENT (fact), COMMAND (intent), or ATTESTATION
entry.actorId = "system";                 // actor identity string
entry.actorType = ActorType.SYSTEM;       // HUMAN, AGENT, or SYSTEM
entry.actorRole = "MyRole";               // descriptive role, e.g. "AdverseEventReporter"
entry.occurredAt = Instant.now();         // not auto-set — must be supplied explicitly
// domain-specific fields...
ledgerRepo.save(entry);
```

### Computing sequenceNumber
The `sequenceNumber` is the position of this entry in the per-subject chain (1-based). casehub-ledger does **not** auto-increment it. Caller pattern:

```java
int nextSeq = ledgerRepo.findLatestBySubjectId(subjectId)
    .map(e -> e.sequenceNumber + 1)
    .orElse(1);
```

### Key surprises within the undocumented
- `id` is not auto-assigned — unlike most Panache entities, `LedgerEntry` has no `@PrePersist` that sets `id`. Call `UUID.randomUUID()` manually.
- `occurredAt` is also not auto-set — must be set explicitly before `save()`.
- `entryType`, `actorType` are enums from `io.casehub.ledger.api.model` (the API module), not `io.casehub.ledger.runtime.model`.

`casehub-ledger/runtime/src/test/java/io/casehub/ledger/privacy/LedgerPrivacyWiringIT.java` — the private `entry()` helper method at the bottom of the class.
